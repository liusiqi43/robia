Detailed Description

template<typename VERTEX, typename EDGE, typename VERTEX_CONSTRUCTOR = DefaultConstructor<VERTEX>, typename EDGE_CONSTRUCTOR = DefaultConstructor<EDGE>, int VERTEX_CHUNK = 1000, int EDGE_CHUNK = 1000>
class vq2::Graph< VERTEX, EDGE, VERTEX_CONSTRUCTOR, EDGE_CONSTRUCTOR, VERTEX_CHUNK, EDGE_CHUNK >

This is an unoriented graph manager.

Author
Herve Frezza-Buet

VERTEX is the value type of the vertex labels (= has to work).

EDGE is the value type of the edge labels (= has to work).

VERTEX_CHUNK and EDGE_CHUNK are the sizes of memory chunks used for allocating vertexes and edges.

VERTEX_CONSTRUCTOR and EDGE_CONSTRUCTOR ar functors for initializing the newly allocated data from an existing one. See DefaultConstructor as an example.

Examples:
example-002-001-graph.cc, example-002-002-graph.cc, example-002-003-distance.cc, example-002-004-connected-components.cc, example-002-005-xfig.cc, example-003-001-closest.cc, example-004-001-kmeans.cc, example-004-002-som.cc, example-004-003-gngt.cc, example-004-004-gngt.cc, example-005-001-speed-som.cc, and example-005-002-speed-gngt.cc.
